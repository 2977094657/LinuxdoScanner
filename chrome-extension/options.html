Windows 开机启动

保存后会在当前 Windows 用户的登录自启动目录里写入一个启动脚本。下次登录系统时，它会自动拉起本地 bridge-server；如果你同时启用了“唤醒浏览器”，还会顺带启动 Chrome / Edge，让扩展更快恢复工作。
